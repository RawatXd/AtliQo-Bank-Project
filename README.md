# AtliQo Bank Project

## 📌 Project Description

**AtliQo Bank** is a fictional banking institution used as a case study for this end-to-end data analysis project. The goal of this project is to perform **Exploratory Data Analysis (EDA)** and derive actionable **business insights** from banking and customer data to support strategic decision-making.

This project is divided into **two phases**:

- **Phase 1 (`main.ipynb`)** — Data Cleaning, Preprocessing & Initial Exploratory Analysis
- **Phase 2 (`phase_2_atliqo_bank.ipynb`)** — Deep-Dive Analysis, Visualizations & Business Insights

The analysis covers key banking metrics such as customer segmentation, transaction behavior, loan patterns, and financial KPIs that help the bank understand its customer base and identify areas of growth or risk.

---

## 🎯 Objectives

- Clean and preprocess raw banking data for analysis
- Perform exploratory data analysis (EDA) to uncover patterns and trends
- Identify key customer segments based on financial behavior
- Analyze loan/transaction data to highlight risk factors and opportunities
- Visualize findings using clear and informative charts
- Provide data-driven business recommendations

## 🚀 How to Run This Project

### 1. Clone the Repository
```bash
git clone https://github.com/RawatXd/AtliQo-Bank-Project.git
cd AtliQo-Bank-Project
```

### 2. Install Required Libraries
```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### 3. Launch Jupyter Notebook
```bash
jupyter notebook
```

### 4. Run the Notebooks in Order
- Open and run **`main.ipynb`** first (Phase 1)
- Then open and run **`phase_2_atliqo_bank.ipynb`** (Phase 2)
